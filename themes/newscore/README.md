# NewsCore Theme

قالب إخباري شامل وحديث لنظام NewsCore.

## الميزات

- ✅ **دعم كامل للغة العربية (RTL)**
- ✅ **الوضع الداكن** - تبديل تلقائي ويدوي
- ✅ **تخطيطات متعددة للصفحة الرئيسية** - كلاسيكي، مجلة، شبكي
- ✅ **تصميم متجاوب** - يعمل على جميع الأجهزة
- ✅ **قائمة Mega Menu** - قوائم ضخمة مع محتوى ديناميكي
- ✅ **أقسام متعددة** - فيديو، بودكاست، بث مباشر، معرض صور
- ✅ **ودجات متعددة** - طقس، عملات، أكثر قراءة، استطلاعات
- ✅ **SEO محسّن** - بيانات منظمة وعلامات meta
- ✅ **أداء عالي** - تحميل كسول وتحسين الصور
- ✅ **إمكانية الوصول** - متوافق مع معايير WCAG

## التثبيت

### 1. نسخ ملفات القالب

```bash
# القالب موجود بالفعل في
/opt/NewsCore/NewsCore-frontend/themes/newscore/
```

### 2. تشغيل Seed لتسجيل القالب

```bash
cd /opt/NewsCore
npx prisma db seed
```

### 3. تفعيل القالب من لوحة التحكم

انتقل إلى: **المظهر > القوالب > NewsCore Theme > تفعيل**

## هيكل القالب

```
themes/newscore/
├── theme.json              # إعدادات القالب
├── README.md               # هذا الملف
├── templates/              # قوالب الصفحات
│   ├── home-magazine.tsx   # قالب الصفحة الرئيسية (مجلة)
│   ├── home-grid.tsx       # قالب الصفحة الرئيسية (شبكي)
│   ├── article-full.tsx    # قالب المقال (عرض كامل)
│   └── index.ts            # تصدير القوالب
├── screenshots/            # لقطات شاشة للقالب
└── previews/               # معاينات القوالب
```

## القوالب المتاحة

### الصفحة الرئيسية

| القالب | الوصف |
|--------|-------|
| `home-classic` | التخطيط الكلاسيكي مع مقال مميز وأخبار جانبية |
| `home-magazine` | تصميم المجلات مع صور بطولية كبيرة |
| `home-grid` | تخطيط شبكي مع أحجام متعددة للمقالات |

### صفحات المقالات

| القالب | الوصف |
|--------|-------|
| `article-default` | صفحة مقال قياسية مع شريط جانبي |
| `article-full-width` | مقال بعرض كامل بدون شريط جانبي |
| `article-featured` | مقال مع صورة بطولية كبيرة |

### الصفحات الثابتة

| القالب | الوصف |
|--------|-------|
| `page-default` | الصفحة الافتراضية |
| `page-full-width` | صفحة بعرض كامل |
| `page-landing` | صفحة هبوط تسويقية |
| `page-contact` | صفحة اتصل بنا |
| `page-about` | صفحة من نحن |

## المكونات المتاحة

### Header & Navigation
- `Header` - الترويسة الرئيسية
- `TopBar` - الشريط العلوي (تاريخ، طقس، عملات)
- `SearchBar` - شريط البحث المنبثق
- `MegaMenu` - القائمة الكبيرة

### Homepage Sections
- `HeroSection` - القسم الرئيسي (3 تخطيطات)
- `BreakingNews` - شريط الأخبار العاجلة
- `CategorySection` - قسم التصنيف
- `VideoSection` - قسم الفيديو
- `OpinionSection` - قسم الرأي والمقالات
- `FeaturesSection` - قسم التقارير والتحقيقات
- `PhotoGallery` - معرض الصور
- `PodcastSection` - قسم البودكاست
- `LiveSection` - قسم البث المباشر
- `PartnersSection` - قسم الشركاء
- `SocialHub` - مركز التواصل الاجتماعي
- `NewsletterSection` - قسم النشرة الإخبارية
- `AppsSection` - قسم التطبيقات

### Widgets
- `MostReadWidget` - الأكثر قراءة
- `NewsletterWidget` - نموذج الاشتراك
- `PollWidget` - استطلاع الرأي
- `WeatherWidget` - ودجة الطقس
- `CurrencyTicker` - شريط العملات

### Utility
- `BackToTop` - زر العودة للأعلى
- `ChatWidget` - نافذة الدردشة
- `CookieNotice` - إشعار الكوكيز
- `LanguageSwitcher` - مبدل اللغة

## التخصيص

### من لوحة التحكم

انتقل إلى **المظهر > تخصيص** لتغيير:

- الألوان (الأساسي، الثانوي، الخلفية، النص)
- الخطوط (نوع الخط، الحجم)
- التخطيط (عرض المحتوى، موقع الشريط الجانبي)
- الترويسة (ثابتة، إظهار/إخفاء عناصر)
- الصفحة الرئيسية (تخطيط البانر، إظهار/إخفاء أقسام)
- صفحة المقال (إظهار/إخفاء عناصر)
- التذييل (عدد الأعمدة، نص حقوق النشر)
- الوضع الداكن (تفعيل، ألوان مخصصة)

### من الكود

عدّل ملف `theme.json` مباشرة لتغيير الإعدادات الافتراضية.
